Tribute to Raja Ravi Varma:
This is a responsive and visually appealing tribute webpage dedicated to Raja Ravi Varma, one of India's most celebrated painters.

Live Link:https://abinaya-balu.github.io/OIBSIP_task2/

Features:

    Elegant Design: Uses soft gradient backgrounds for an artistic feel.

    Interactive Elements: Hover effects on artwork images for a dynamic experience.

    Informative Content: Covers the artist's biography, legacy, and admired works.

    Grid Layout: Displays famous paintings beautifully in a structured format.

Technologies Used:

    HTML: Structuring the tribute page.

    CSS: Styling for aesthetic appeal and smooth animations.

    JavaScript (optional): Could be extended for interactivity.

How to Use:

    Open the index.html file in a web browser.

    Scroll through the detailed tribute content.

    Read about Raja Ravi Varma's artistic contributions.
